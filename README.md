# Bank Account Management System 🏦

> An internship project developed under **Upskill Campus**, demonstrating real-world banking operations with a graphical user interface and robust database persistence.

This desktop application provides a secure and structured way to manage bank accounts, handle financial transactions, and maintain data integrity. It utilizes a Java Swing frontend for a seamless user experience and Hibernate ORM for efficient database management.

---

## 🚀 Key Features

* **Graphical User Interface:** An intuitive, easy-to-use desktop interface built with Java Swing.
* **Account Management:** Register new users, generate unique account numbers, and manage user profiles.
* **Transaction Processing:** Securely handle deposits, withdrawals, and fund transfers with real-time balance validation.
* **Data Persistence:** Reliable data storage and retrieval using Hibernate ORM, ensuring transaction histories and account balances are permanently saved.
* **Balance & History Tracking:** Allow users to instantly check their current available funds and view past transaction logs.

---

## 🛠️ Tech Stack

* **Core Language:** Java
* **GUI Framework:** Java Swing
* **Object-Relational Mapping (ORM):** Hibernate
* **Build Tool & Dependency Management:** Maven (`pom.xml`)
* **IDE:** IntelliJ IDEA
